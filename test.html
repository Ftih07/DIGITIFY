Pertanyaan Seputar Layanan Kami

Jawaban ringkas untuk hal-hal yang paling sering ditanyakan (FAQ)

Apa maksudnya "Paket Terima Beres"?
Anda tidak perlu pusing memikirkan urusan teknis seperti sewa server, konfigurasi domain, atau hal rumit lainnya. Anda cukup siapkan materi teks dan foto produk/bisnis, lalu kami yang akan mengerjakan semuanya sampai website siap digunakan dan online di internet.
Apakah ada biaya perpanjangan per bulan/tahun?
Tidak ada biaya bulanan. Namun, selayaknya menyewa lahan untuk bisnis, ada biaya perpanjangan tahunan untuk menjaga nama domain dan server tetap aktif. Biayanya sangat terjangkau, mulai dari Rp100.000 hingga Rp500.000 per tahun, tergantung paket yang Anda pilih di awal.
Berapa lama proses pembuatan websitenya?
Untuk Paket Web Template, pengerjaan sangat cepat (bisa selesai dalam 1 hari) setelah materi lengkap kami terima. Sedangkan untuk Paket Custom dan Sistem CMS biasanya memakan waktu 3 hingga 7 hari kerja karena butuh penyesuaian desain dan perancangan sistem database.
Apakah saya bisa mengubah isi website sendiri nanti?
Sangat bisa, asalkan Anda memilih Paket Web + Sistem CMS atau paket E-Commerce. Anda akan mendapatkan akses ke Dashboard Admin yang sangat mudah digunakan (seperti memakai media sosial) untuk menambah artikel, mengubah harga, atau meng-update foto promo secara mandiri.
Saya masih bingung memilih paket, bagaimana ya?
Jangan ragu untuk bertanya! Silakan klik tombol Konsultasi Proyek untuk terhubung dengan kami via WhatsApp. Ceritakan saja model bisnis Anda, nanti kami bantu pilihkan solusi digital yang paling pas dengan budget dan kebutuhan Anda.
